##  Deskripsi Aplikasi
Aplikasi sederhana Android menggunakan Kotlin yang bisa:
- Menerima input nama
- Menampilkan sapaan "Hello {nama yang dimasukkan}"
- Mengubah tema mode terang dan mode gelap

## Cara Menjalankan
1. Clone repository ini
2. Buka di Android Studio
3. Cari Icon "Run"
4. Klik Tombol Run
